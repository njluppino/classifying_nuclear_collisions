# classifying_nuclear_collisions

###############################

SOURCE CODE DESCRIPTION

###############################

Neural Network i wrote to classify whether a nuclear collision produced a jet based on training data that includes the collisions various parameters.
Code is written in python using keras. Have a few different models as a comparison seeing how the different methods effect the models accuracy.
Hidden layers use a mix of activation functions like tanh(x) and simgoid.

##################################

PRESENTATION DESCRIPTION

##################################

Presentation to walk through each piece of the code and describe the different methods used to increase the models accuracy.
